### Lecture 1
```
По поводу вчерашней лекции:
1) вот репозиторий, с тем, что делали вчера https://github.com/pavel06081991/react-presentation.git

git clone https://github.com/pavel06081991/react-presentation.git
если yarn не установлен глобально, устанавливаем его, в консоле пишем npm i yarn -g
в консоли запускаем команду yarn
дальше в консоли запускаем команду yarn run start
дожидаемся выполнения предыдущей команды и открываем браузер с урлом localhost:3000
Компоненты, которые писали, находятся в папке app/components
Точка входа(метод ReactDOM.render) находится в файле app/app.js

По поводу boilerplate-ов, вот они, два самых популярных:

https://github.com/react-boilerplate/react-boilerplate
https://github.com/kriasoft/react-starter-kit
в репозиториях boilerplate-ов обычно всегда описано, как с ними работать
По поводу стилей внутри javascript-а. Вот та библиотека, о которой говорил - https://www.styled-components.com

Что такое css modules и зачем они нужны описано вот в этом репозитории, который является своего рода спецификацией для css modules - https://github.com/css-modules/css-modulest
```
